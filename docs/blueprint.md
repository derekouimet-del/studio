# **App Name**: ProSentry

## Core Features:

- Network Scan: Conduct network scans to identify live hosts, open ports, and running services.
- Vulnerability Assessment: Assess identified services and hosts for known vulnerabilities using a local database or a third party api with LLM powered reasoning.
- Exploit Suggester: Suggest potential exploits for identified vulnerabilities. The LLM reasoning tool can assess exploit suitability.
- Reporting: Generate reports with findings and remediation recommendations.
- Live Monitoring: Monitor network traffic in real-time to detect anomalies or suspicious activity.
- Web Crawling: Crawl the pages of specified URL and look for credentials

## Style Guidelines:

- Primary color: Deep Blue (#2E3192) to convey professionalism and security.
- Background color: Dark Gray (#2D2D2D) for a modern, focused interface.
- Accent color: Electric Blue (#7DF9FF) to highlight important elements and actions.
- Body and headline font: 'Inter' (sans-serif) for readability and a clean, modern appearance.
- Code font: 'Source Code Pro' for displaying code snippets and scan results.
- Use sharp, modern icons to represent tools and actions.
- Subtle animations to indicate loading and activity, maintaining a professional feel.
